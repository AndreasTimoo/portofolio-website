My Project

Interactive Map of Clinic Ar-Rahim Patient Distribution

This project visualizes the distribution of patients at Klinik Ar-Rahim using an interactive map built with Leaflet.js. The data, sourced from QGIS and provided in GeoJSON format, categorizes patient numbers with a color-coded legend for better analysis. The map is hosted on GitHub Pages for easy access and public viewing.
